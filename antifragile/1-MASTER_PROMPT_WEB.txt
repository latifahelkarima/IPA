## ARSITEKTUR APLIKASI
Aplikasi saya menggunakan Vanilla JS dan mengandalkan file JSON untuk data materi dan kuis.
Setiap materi baru yang dibuat akan diletakkan di dalam folder `materi/[slug-materi]/`.
Dalam folder materi tersebut wajib ada 3 file:
1. `materi.json` (Isi materi yang dipecah jadi konsep-konsep kecil/Progressive Disclosure).
2. `quiz.json` (Soal kuis interaktif berdasarkan materi).
3. `index.html` (Pemanggil template materi).
Lalu file `data/materi-list.json` di root aplikasi harus diupdate agar materi baru muncul di halaman depan.

## ATURAN KONTEN (WAJIB DIIKUTI)
- **Matematika/Rumus**: Gunakan format KaTeX `$$rumus$$` untuk block, dan `$rumus$` untuk inline.
- **Gambar**: Jika materi butuh ilustrasi gambar, beri tahu saya untuk mengupload gambar ke postimages.org, lalu tuliskan kode markdown `<img src="[URL_GAMBAR]" style="width:100%; border-radius:12px; margin:16px 0;">` di dalam `materi.json`.
- **Format JSON**: Pastikan JSON valid, tidak ada koma di akhir array/object yang salah.
- **Konsep Kecil**: Pecah materi menjadi maksimal 5-7 konsep agar siswa tidak bosan.

**LANGKAH 2: Simpan Materi (materi.json)**
Buat file `materi.json` di dalam folder `materi/[slug-materi]/` dan paste kode ini:
```json
{
  "title": "Judul",
  "description": "Deskripsi",
  "theme": "math",
  "concepts": [
    {
      "id": "konsep-1",
      "title": "Judul Konsep",
      "icon": "menu_book",
      "content": "Isi materi. Gunakan **tebal** atau *miring* atau tag <img src='...'> jika butuh gambar.",
      "steps": ["Poin 1", "Poin 2"],
      "summary": "Ringkasan"
    }
  ]
}
```

**LANGKAH 3: Simpan Kuis (quiz.json)**
Buat file `quiz.json` di dalam folder `materi/[slug-materi]/` dan paste kode ini:
```json
{
  "quizTitle": "Kuis: [Judul Materi]",
  "questions": [
    {
      "id": 1,
      "question": "Pertanyaan 1?",
      "options": ["A", "B", "C", "D"],
      "answer": "B",
      "explanation": "Penjelasan."
    }
  ]
}
```
*(Bebas tambahkan soal sebanyak apa pun).*

**LANGKAH 4: Munculkan di Beranda (materi-list.json)**
Buka file `data/materi-list.json`. Tambahkan blok JSON ini ke dalam array `"materi": [...]` sebelum kurung tutup `]`. Jangan lupa tambahkan koma `,` di objek sebelumnya!
```json
    {
      "id": "[slug-materi]",
      "slug": "[slug-materi]",
      "title": "[Judul Materi]",
      "description": "[Deskripsi Singkat]",
      "category": "math",
      "thumbnail": "",
      "totalConcepts": 5,
      "estimatedTime": "15 menit",
      "difficulty": "mudah",
      "tags": ["tag1"],
      "published": true
    }
```

Selesai! Buka browser dan refresh `localhost`.
