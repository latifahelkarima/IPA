# Source Materials

Folder ini adalah tempat Anda menaruh materi mentah (raw materials) sebelum diubah oleh AI menjadi sistem pembelajaran interaktif.

## Cara Penggunaan untuk Rekan Guru:

1. Buat file baru di sini (misal: `materi-1-sejarah.md` atau `teks-bacaan.txt`).
2. Masukkan rangkuman materi, teks buku cetak, atau coretan konsep Anda ke dalam file tersebut.
3. Buka AI Assistant Anda (Claude/ChatGPT/Gemini) di dalam project ini.
4. Salin isi dari file `prompts/ingest-material.md` dan kirimkan ke AI dengan memberitahu lokasi materi mentah Anda.
5. Biarkan AI yang bekerja memecah materi Anda menjadi konsep kecil (Progressive Disclosure), membuat kuis, dan membangun halaman interaktifnya secara otomatis!
